#  Exploratory Data Analysis (EDA Basic) Project

##  Project Overview
This project performs basic Exploratory Data Analysis (EDA) on the House Price India dataset using Python. The objective is to understand the structure of the dataset, identify trends, and extract meaningful insights through statistical analysis and visualization.

The project demonstrates practical data analysis using a real-world housing dataset.

---

##  Dataset
- Dataset Used: **House Price India Dataset**
- File: `House Price India.csv`

The dataset contains house-related features such as:
- Number of Bedrooms  
- Number of Bathrooms  
- Living Area  
- Lot Area  
- Floors  
- Condition  
- Price  

---

##  Objectives
- Understand dataset structure  
- Perform statistical analysis  
- Explore feature distributions  
- Analyze relationships between variables  
- Visualize important patterns in house prices  

---

##  EDA Techniques Performed

### 1. Data Inspection
- Dataset shape  
- Data types  
- Summary statistics  

### 2. Univariate Analysis
- Distribution of house prices  
- Distribution of bedrooms and bathrooms  

### 3. Bivariate Analysis
- Price vs Living Area  
- Price vs Number of Bedrooms  
- Price vs Bathrooms  

### 4. Correlation Analysis
- Heatmap of numerical variables  

### 5. Visualization
- Histograms  
- Scatter Plots  
- Bar Charts  
- Boxplots  
- Heatmaps  

---

##  Technologies Used

- Pandas

- Matplotlib  
- Seaborn

##  Key Insights
- Houses with larger living area tend to have higher prices  
- Number of bedrooms and bathrooms positively affect price  
- Certain features show strong correlation with house prices  
- Outliers are present in price distribution  

---

##  Conclusion
EDA helped uncover major factors affecting house prices and identify important patterns in the housing dataset. Visualization techniques made the analysis more understandable and insightful.
