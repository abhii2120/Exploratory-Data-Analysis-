#  Exploratory Data Analysis (EDA) Repository

##  Overview
This repository contains multiple Exploratory Data Analysis (EDA) projects completed using Python and real-world datasets. The projects focus on understanding data, discovering patterns, identifying relationships, and generating meaningful insights through statistical analysis and visualization.

These projects demonstrate practical data analysis and visualization skills used in real-world scenarios.

---

##  Repository Structure

### 🔹 EDA Basic
Basic Exploratory Data Analysis performed on the **House Price India Dataset**.

Includes:
- Data Inspection  
- Statistical Summary  
- Univariate Analysis  
- Bivariate Analysis  
- Correlation Analysis  
- Basic Visualizations  

---

### 🔹 EDA Visualization
Advanced visualization-based EDA performed on the **Titanic Dataset**.

Includes:
- Survival Analysis  
- Passenger Class Analysis  
- Gender-based Analysis  
- Correlation Heatmaps  
- Advanced Data Visualization
