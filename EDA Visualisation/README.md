#  Exploratory Data Analysis (EDA) Visualization Project

##  Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python. The goal is to analyze the dataset, discover patterns, identify relationships, and visualize important insights through graphical representations.

The project demonstrates practical data analysis skills using a real-world dataset.

---

##  Dataset
- Dataset Used: **Titanic Dataset**
- File: `Titanic.csv`

The dataset contains passenger information such as:
- Age  
- Gender  
- Passenger Class  
- Fare  
- Survival Status  
- Embarkation Port  

---

##  Objectives
- Understand dataset structure and summary  
- Perform univariate and bivariate analysis  
- Identify patterns and trends in data  
- Visualize relationships between variables  
- Generate insights from the Titanic dataset  

---

##  EDA Techniques Performed

### 1. Data Inspection
- Dataset shape  
- Data types  
- Summary statistics  

### 2. Univariate Analysis
- Distribution of Age  
- Fare Distribution  
- Passenger Class Analysis  

### 3. Bivariate Analysis
- Survival vs Gender  
- Survival vs Passenger Class  
- Fare vs Survival  

### 4. Correlation Analysis
- Relationship between numerical variables  

### 5. Visualization
- Histograms  
- Bar Charts  
- Boxplots  
- Countplots  
- Heatmaps  

---

##  Key Insights
- Female passengers had a higher survival rate  
- First-class passengers had better survival chances  
- Younger passengers showed different survival trends  
- Fare and passenger class influenced survival probability  

---

##  Conclusion
Exploratory Data Analysis helped uncover important patterns and insights from the Titanic dataset. Visualization techniques made it easier to understand relationships between variables and derive meaningful conclusions.
